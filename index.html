CSS Grid

and flexbox

From pure HTML to complex layouts with CSS Flexbox and Grid Layout, the web certainly looks very different today than the 1990s. CSS technologies have changed over time, though Flexbox and Grid represent a point in its history that makes it possible for web designers/developers to create layouts that reflect graphic design principles such as overlapping in an easy and flexible way.

Flexbox and Grid are able to work in conjunction. For example, developers might make all page elements part of the Grid, but make one of those grid elements into a flex container. While Grid can position content in both dimensions, Flexbox can be useful for aligning Grid elements in one dimension.
